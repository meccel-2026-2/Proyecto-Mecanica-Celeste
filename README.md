# Proyecto-Mecanica-Celeste
Notebook realizado para el proyecto del curso de Mecanica Celeste.

## Configuracion como proyecto Python

### 1) Crear y activar entorno virtual (Windows PowerShell)

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 2) Instalar dependencias

```powershell
pip install -r requirements.txt
```

### 3) Instalacion editable del proyecto (opcional)

```powershell
pip install -e .
```
